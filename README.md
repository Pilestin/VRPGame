# VRPGame

An interactive VRP game platform where players solve routing problems (VRP, CVRP, CEVRP, CEVRPTW) by connecting nodes on a map.

TR: Web üzerinde oynanan, VRP tabanlı bir puzzle/oyun. Oyuncu harita üzerindeki düğümleri (depo, müşteri, şarj istasyonu) birleştirerek geçerli ve mesafe maliyeti açısından iyi rotalar üretmeye çalışır. Oyun; VRP, CVRP, CEVRP, CEVRPTW modlarını destekler (checkbox ile seçilebilir). ESOGU-CVRP veri kümeleri ve Routing Markup Language (RML) dosyalarıyla çalışır.

>  Not: Bu proje henüz geliştirme aşamasındadır. Şimdilik sadece TSP modu desteklenmektedir.

### Gelecek çalışmalarda

- VRP ve farklı modlarının (CVRP, CEVRP, CEVRPTW) desteklenmesi
- Random problemler yerine gerçek literatür test setlerinin kullanılması
- Test için RML dosyalarının yüklenebilmesi veya uygulama üzerinde settings menüsünden ayarların yapılabilmesi. 
- Farklı zorluk seviyeleri. Rotalama kısıtlarının (kapasite, zaman penceresi, elektrikli araç kısıtları vb.) eklenmesi
- Oyuncu etkileşiminin artırılması için daha fazla görsel ve sesli geri bildirim. Oyun içi animasyonlar ve ses efektleri 
- ESOGU dataseti -> arka planda gerçek harita entregrasyonu
- Oyuncu performansının kaydedilmesi ve liderlik tablosu
- Oyun içi ipuçları ve rehberlik sisteminin eklenmesi
